# 📦 Inventory Demand Forecasting System

An AI-powered inventory analytics system that predicts future product demand, identifies stockout risks, detects unusual sales patterns, and recommends optimal reorder quantities to improve inventory management.

---

## 🚀 Features

- 📈 Demand Forecasting using Machine Learning
- ⚠️ Stockout Risk Detection
- 📊 Sales Trend Analysis
- 🔍 Anomaly Detection in Sales Data
- 📦 Intelligent Reorder Quantity Recommendation
- 📉 Inventory Performance Insights

## ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/bhavyaanil07/Inventory_Demand_Forecasting_System.git
```

Move into the project

```bash
cd Inventory_Demand_Forecasting_System
```

Install dependencies

```bash
pip install -r requirements.txt
```

## ▶️ Usage

Run the notebook
